# Bank of America Customer Churn – SQL Case Study (Synthetic Data)

## Business Question
Banks lose money when customers close accounts.  
This case study analyzes a synthetic Bank of America customer dataset to identify:
- churn rate overall
- which customer segments churn the most
- retention insights the bank can act on

## Dataset
Synthetic (fake but realistic) customer dataset modeled after a U.S. retail bank.
No real Bank of America data is used.

## What This Project Includes
- SQL table schema + data import
- Churn KPIs
- Customer segmentation analysis
- Retention recommendations

## Repository Structure
- `data/` → synthetic dataset CSV  
- `sql/` → all SQL queries + analysis  
- `screenshots/` → query outputs  

## Skills Demonstrated
SQL, JOINs, GROUP BY, CASE, KPIs, customer analytics, banking domain insights.
